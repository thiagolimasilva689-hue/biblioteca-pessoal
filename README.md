# Biblioteca Pessoal - CSV + SQLite

Projeto de pipeline de dados que lê um arquivo CSV e importa para banco SQLite.

## Funcionalidades
- Importar livros de CSV para SQLite
- Consultar livros por autor
- Atualizar e excluir registros
- Relatório com COUNT, AVG, mais antigo, mais novo

## Tecnologias
- Python 3
- SQLite3
- CSV

## Como usar
1. Execute: `python biblioteca.py`
